BolaKu

BolaKu merupakan website penyedia informasi seputar sepak bola yang ada di Eropa dengan 6 Competitions teratas. BolaKu ini di bangun menggunakan framework Materialize Css dan juga dengan Menggunakan Progressive Web App

Progressive Web App Merupakan sebuah aplikasi yang dibangun dengan melakukan optimasi pada sebuah website. Optimasi yang dilakukan tidak hanya akan membuat website menjadi lebih cepat namun juga mampu memberikan pengalaman layaknya menggunakan aplikasi mobile.

Selain itu, Anda juga tidak perlu membuat aplikasimobile secara terpisah. Beberapa contoh website yang menggunakan PWA adalah Twitter, Pinterest, OLX dan Trivago. PWA merupakan aplikasi hybrid yang terinspirasi dari kemudahan penggunaan browser di berbagai platform dan kenyamanan menggunakan aplikasi mobile di ponsel atau tablet. Progressive Web App sendiri juga bisa berjalan saat offline tergantung data yang sudah ter cache sebelumnya
